# Host an A2A Server

## ¿Qué es?
- Publica un agente **A2A** para que clientes y otros agentes puedan acceder a él mediante **HTTP**.
- Actúa como puente entre la lógica del agente y el mundo externo.

## ¿Qué permite?
- 🌐 Publicar el **Agent Card**.
- 📩 Recibir solicitudes A2A.
- ⚙️ Enviarlas al **Agent Executor**.
- 🔄 Gestionar el ciclo de vida de las tareas (streaming, estado, cancelación).

## Componentes principales

### Agent Card
Describe el agente.

- Capacidades.
- Skills.
- Modos de entrada/salida.
- Se expone normalmente en:

```text
/.well-known/agent-card.json
```

### Request Handler
Intermediario entre el servidor y el **Agent Executor**.

Funciones:
- Enrutar solicitudes (`execute`, `cancel`).
- Administrar tareas mediante un **Task Store**.

### Task Store
Almacena el estado de las tareas.

- Seguimiento de tareas.
- Streaming.
- Reanudación de suscripciones.
- Gestión del ciclo de vida.

### Server Application
Aplicación web que expone el agente.

Tecnologías comunes:
- **Starlette** → framework ASGI.
- **Uvicorn** → servidor ASGI.

## Flujo de funcionamiento
```text
Cliente / Otro agente
        │
        ▼
HTTP Server (Starlette + Uvicorn)
        │
        ▼
Request Handler
        │
        ▼
Agent Executor
        │
        ▼
Lógica del agente
        │
        ▼
Respuesta
```

## Pasos para alojar un agente
1. Definir **Skills**.
2. Crear el **Agent Card**.
3. Configurar el **Request Handler**.
4. Conectar el **Task Store**.
5. Crear la aplicación del servidor.
6. Ejecutar el servidor con **Uvicorn**.
7. El agente queda disponible para recibir solicitudes.

## Ejemplo
Un agente **Hello World**:

- Publica una habilidad de saludo.
- Expone su **Agent Card**.
- Escucha solicitudes HTTP.
- Devuelve `"Hello World"` cuando un cliente lo invoca.

## Idea clave
- **Agent Card** → describe el agente.
- **Request Handler** → dirige las solicitudes.
- **Agent Executor** → ejecuta la lógica.
- **Task Store** → administra el estado de las tareas.
- **Server (Starlette + Uvicorn)** → publica el agente en la red.
