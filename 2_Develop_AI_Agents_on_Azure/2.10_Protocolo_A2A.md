# Define un A2A Agent

## ¿Qué es A2A?
- Protocolo estándar para que agentes de IA se comuniquen y colaboren.
- Permite compartir contexto, invocar capacidades e intercambiar información de forma segura.
- Facilita la interoperabilidad entre agentes de diferentes plataformas y proveedores.

## Ventajas de A2A
- 🤝 **Colaboración:** agentes pueden trabajar juntos entre distintos sistemas.
- 🧠 **Modelo flexible:** cada agente puede usar el LLM que mejor se adapte a su tarea.
- 🔒 **Autenticación integrada:** comunicación segura entre agentes.

## Agent Skill
Describe una capacidad específica del agente.

### Componentes
- **ID:** identificador único.
- **Name:** nombre de la habilidad.
- **Description:** explicación de la función.
- **Tags:** palabras clave para descubrimiento.
- **Examples:** ejemplos de uso.
- **Input/Output Modes:** formatos soportados (texto, JSON, etc.).

> Una habilidad representa una tarea concreta que el agente puede realizar.

## Agent Card
Documento que permite a otros agentes descubrir y utilizar el agente.

### Componentes
- **Identity:** nombre, descripción y versión.
- **Endpoint URL:** dirección del servicio A2A.
- **Capabilities:** funciones soportadas (streaming, notificaciones, etc.).
- **Default Input/Output Modes:** formatos por defecto.
- **Skills:** habilidades disponibles.
- **Authentication:** método de autenticación requerido.

> Es el equivalente a una **tarjeta de presentación** del agente.

## Flujo de funcionamiento
1. El agente define sus **Skills**.
2. Publica su **Agent Card**.
3. Otros agentes descubren sus capacidades.
4. Las solicitudes se enrutan al **Skill** correspondiente.
5. El agente responde en el formato soportado.

## Ejemplo
```text
Agente A (Generar títulos)
        ↓
Routing Agent (lee Agent Cards)
        ↓
Agente B (Generar outline)
        ↓
Artículo completo
```

## Idea clave
- **Agent Skill = lo que el agente sabe hacer.**
- **Agent Card = cómo otros agentes descubren y utilizan esas habilidades.**
- **A2A = protocolo que hace posible la colaboración entre agentes.**
