# ¿Qué son las herramientas (Tools)?

Las **Tools** permiten que un modelo de IA interactúe con sistemas externos para realizar tareas que van más allá de generar texto. En lugar de responder únicamente con el conocimiento adquirido durante su entrenamiento, el modelo puede obtener información actualizada, ejecutar código, consultar archivos o llamar funciones externas. :contentReference[oaicite:0]{index=0}

---

# ¿Por qué usar Tools?

Los modelos de lenguaje tienen limitaciones:

- Su conocimiento puede estar desactualizado.
- No pueden acceder directamente a Internet.
- No pueden ejecutar código por sí mismos.
- No pueden consultar bases de datos o archivos privados.

Las **Tools** amplían estas capacidades permitiendo que el modelo interactúe con recursos externos. :contentReference[oaicite:1]{index=1}

---

# ¿Cómo funcionan?

Cuando el usuario realiza una solicitud:

1. El usuario envía un **prompt**.
2. El modelo analiza si necesita utilizar una herramienta.
3. Si es necesario, invoca la herramienta adecuada.
4. La herramienta devuelve el resultado.
5. El modelo utiliza esa información para generar la respuesta final.

```text
Usuario
    │
    ▼
 Prompt
    │
    ▼
 Modelo
    │
¿Necesita una Tool?
    │
 ┌──┴─────────────┐
 │ Sí            │ No
 ▼               ▼
Tool         Respuesta
 │
 ▼
Resultado
 │
 ▼
Respuesta final
```

# Tipos de herramientas

## 1. Code Interpreter

Permite ejecutar código en un entorno seguro.

### Capacidades

- Ejecutar Python.
- Analizar datos.
- Realizar cálculos.
- Generar gráficos.
- Procesar archivos.

**Ejemplo:** analizar un archivo CSV y crear un gráfico de ventas.

---

## 2. Web Search

Permite buscar información actualizada en Internet.

### Casos de uso

- Noticias recientes.
- Clima.
- Información en tiempo real.
- Eventos actuales.

**Ejemplo:** consultar el precio actual de una acción.

---

## 3. File Search

Permite buscar información dentro de documentos proporcionados por el usuario.

Puede consultar:

- PDF
- Word
- PowerPoint
- Texto
- Otros documentos compatibles

**Ejemplo:** responder preguntas sobre un manual de 500 páginas sin leerlo completo.

---

## 4. Function Calling (Functions Tool)

Permite que el modelo invoque funciones definidas por el desarrollador para interactuar con sistemas externos.

### Ejemplos

- Consultar una base de datos.
- Obtener el inventario de una tienda.
- Crear una reserva.
- Enviar un correo electrónico.
- Consultar una API.

> El modelo **no ejecuta la lógica directamente**; decide cuándo llamar la función y utiliza el resultado para responder.

---

# Beneficios de las Tools

- Acceso a información actualizada.
- Integración con sistemas externos.
- Automatización de tareas.
- Mayor precisión en las respuestas.
- Posibilidad de realizar acciones, no solo generar texto.

---

# Casos de uso

- Chatbots con información en tiempo real.
- Asistentes empresariales.
- Análisis de documentos.
- Automatización de procesos.
- Integración con APIs y bases de datos.

---

# Resumen

- Las **Tools** amplían las capacidades de un modelo de IA permitiéndole interactuar con recursos externos.
- Microsoft AI Foundry incluye cuatro herramientas principales:
  - **Code Interpreter:** ejecuta código y analiza datos.
  - **Web Search:** obtiene información actualizada de Internet.
  - **File Search:** busca y analiza documentos.
  - **Function Calling:** conecta el modelo con funciones y APIs personalizadas.
- Gracias a las Tools, los modelos pueden **consultar información, ejecutar acciones y ofrecer respuestas más útiles y precisas**.
