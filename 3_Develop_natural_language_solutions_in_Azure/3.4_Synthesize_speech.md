# Synthesize Speech (Text-to-Speech)

## ¿Qué es?
Convierte **texto** en **audio** mediante un modelo de síntesis de voz.

## Modelos compatibles
- `gpt-4o-tts`
- `gpt-4o-mini-tts`

> La disponibilidad depende de la región de Azure.

## Flujo de funcionamiento

```text
Texto
   │
   ▼
AzureOpenAI Client
   │
   ▼
Modelo TTS
   │
   ▼
Audio (.wav)
```

## Pasos de uso

1. Crear un **AzureOpenAI Client**.
2. Conectarse al recurso Foundry.
3. Enviar el texto al modelo TTS.
4. Elegir una **voz** (`voice`).
5. (Opcional) Indicar el estilo mediante `instructions`.
6. Guardar el audio generado.

## Componentes principales

### AzureOpenAI
Cliente para acceder a los modelos de Azure OpenAI.

### audio.speech.create()
Genera audio a partir de texto.

Parámetros principales:
- `model` → modelo TTS.
- `input` → texto a convertir.
- `voice` → voz utilizada.
- `instructions` → estilo o tono de la voz.

## Personalización

Ejemplo:

```text
Texto:
"This speech was AI-generated!"

Voice:
alloy

Instructions:
"Speak in an upbeat, excited tone."
```

Resultado:
- Audio generado con la voz **alloy** y un tono entusiasta.

## Idea clave
- **Text-to-Speech (TTS)** convierte texto en voz.
- Se utiliza mediante **AzureOpenAI**.
- Permite elegir la **voz** y controlar el **tono** mediante instrucciones.
- Modelos disponibles: **gpt-4o-tts** y **gpt-4o-mini-tts**.
