# Traducir voz (Speech Translation)

## ¿Qué es?

**Azure Speech Translation** permite traducir voz hablada a otros idiomas.

Puede devolver el resultado como:

- Texto traducido (Speech → Text Translation)
- Voz sintetizada (Speech → Speech Translation)

Todo esto forma parte de **Azure Speech en Microsoft Foundry Tools**.

---

# Flujo general

```text
Usuario habla
        │
        ▼
Speech Recognition
        │
        ▼
Traducción
        │
        ├────────► Texto traduccido
        │
        └────────► Voz sintetizada
```

---

# Componentes principales

| Componente | Función |
|------------|---------|
| `SpeechTranslationConfig` | Configura la traducción de voz. |
| `AudioConfig` | Define el origen del audio (micrófono o archivo). |
| `TranslationRecognizer` | Reconoce la voz y la traduce. |
| `SpeechSynthesizer` | Convierte el texto traducido nuevamente en voz. |

---

# Conectarse al recurso Azure Speech

Se utiliza un objeto:

```python
SpeechTranslationConfig
```

Puede conectarse mediante:

- Endpoint
- Región

## Ejemplo

```python
import azure.cognitiveservices.speech as speech_sdk

# Conectarse mediante endpoint
translation_cfg = speech_sdk.translation.SpeechTranslationConfig(
    subscription="FOUNDRY_KEY",
    endpoint="FOUNDRY_ENDPOINT"
)

# O mediante región
translation_cfg = speech_sdk.translation.SpeechTranslationConfig(
    subscription="FOUNDRY_KEY",
    region="FOUNDRY_REGION"
)
```

---

# Traducir voz a texto

Se utiliza:

```python
TranslationRecognizer
```

## Ejemplo

```python
translator = speech_sdk.translation.TranslationRecognizer(
    translation_config=translation_cfg,
    audio_config=audio_cfg
)

print("Speak now...")

translation_results = translator.recognize_once_async().get()

print(f"Translating '{translation_results.text}'")

translations = translation_results.translations

for translation_language in translations:
    print(
        f"{translation_language}: "
        f"'{translations[translation_language]}'"
    )
```

Salida

```text

Translating 'Hello.'

fr: 'Bonjour.'

ja: 'こんにちは。'
```

# Traducir voz a voz (Speech-to-Speech)

Existen **dos enfoques**.

---

# 1. Manual Synthesis (Síntesis manual)

Es el enfoque más flexible.

Se divide en dos pasos independientes.

## Paso 1

Traducir voz a texto.

```text
Speech
    │
    ▼
Texto traducido
```

---

## Paso 2

Sintetizar cada traducción usando `SpeechSynthesizer`.

```text
Texto traducido
        │
        ▼
Audio
```

---

## Flujo completo

```text
Hablar
      │
      ▼
TranslationRecognizer
      │
      ▼
Texto traducido
      │
      ▼
SpeechSynthesizer
      │
      ▼
Audio
```

---

## Ejemplo completo

```python
import azure.cognitiveservices.speech as speech_sdk

# Configurar traducción
translation_cfg = speech_sdk.translation.SpeechTranslationConfig(
    subscription="FOUNDRY_KEY",
    endpoint="FOUNDRY_ENDPOINT"
)

translation_cfg.speech_recognition_language = "en-US"
translation_cfg.add_target_language("fr")
translation_cfg.add_target_language("ja")

audio_cfg = speech_sdk.AudioConfig(
    use_default_microphone=True
)

# Configurar Speech Synthesis
speech_cfg = speech_sdk.SpeechConfig(
    subscription="FOUNDRY_KEY",
    endpoint="FOUNDRY_ENDPOINT"
)

audio_out_cfg = speech_sdk.audio.AudioOutputConfig(
    use_default_speaker=True
)

voices = {
    "fr": "fr-FR-HenriNeural",
    "ja": "ja-JP-NanamiNeural"
}

translator = speech_sdk.translation.TranslationRecognizer(
    translation_config=translation_cfg,
    audio_config=audio_cfg
)

print("Speak now...")

translation_results = translator.recognize_once_async().get()

translations = translation_results.translations

for translation_language in translations:

    print(
        f"{translation_language}: "
        f"{translations[translation_language]}"
    )

    speech_cfg.speech_synthesis_voice_name = voices.get(
        translation_language
    )

    speech_synthesizer = speech_sdk.SpeechSynthesizer(
        speech_cfg,
        audio_out_cfg
    )

    speak = speech_synthesizer.speak_text_async(
        translations[translation_language]
    ).get()

    if speak.reason != speech_sdk.ResultReason.SynthesizingAudioCompleted:
        print(speak.reason)
```

---

# Ventajas del Manual Synthesis

- Permite traducir a múltiples idiomas.
- Puedes elegir una voz distinta para cada idioma.
- Mayor control sobre la síntesis.
- Compatible con cualquier cantidad de idiomas.

---

# 2. Event-Based Synthesis

Este enfoque automatiza la síntesis.

No necesitas crear un `SpeechSynthesizer`.

La traducción devuelve directamente el audio.

---

## Limitación importante

> **Solo funciona para traducción 1:1.**

Ejemplo válido:

```text
en-US
        │
        ▼
fr
```

No válido:

```text
en-US
        ├──► fr
        ├──► ja
        └──► es
```

---

# Funcionamiento

Se debe:

- Configurar una voz.
- Registrar un evento `Synthesizing`.
- Capturar el audio generado.

---

## Configurar la voz

```python
translation_cfg.voice_name = "fr-FR-HenriNeural"
```

---

# Crear el evento

```python
def synthesis_callback(evt):

    size = len(evt.result.audio)

    print(
        f"Audio synthesized: {size} byte(s)"
    )

    if size > 0:

        file = open(output_file, "wb+")

        file.write(evt.result.audio)

        file.close()
```

---

# Asociar el evento

```python
translator.synthesizing.connect(
    synthesis_callback
)
```

---

# Ejemplo completo

```python
import azure.cognitiveservices.speech as speech_sdk
from playsound3 import playsound

source_language = "en-US"
target_language = "fr"

output_file = "translation.wav"

translation_cfg = speech_sdk.translation.SpeechTranslationConfig(
    subscription="FOUNDRY_KEY",
    endpoint="FOUNDRY_ENDPOINT"
)

translation_cfg.speech_recognition_language = source_language
translation_cfg.add_target_language(target_language)

translation_cfg.voice_name = "fr-FR-HenriNeural"

audio_cfg = speech_sdk.AudioConfig(
    use_default_microphone=True
)

translator = speech_sdk.translation.TranslationRecognizer(
    translation_config=translation_cfg,
    audio_config=audio_cfg
)

def synthesis_callback(evt):

    size = len(evt.result.audio)

    print(
        f"Audio synthesized: {size} byte(s)"
    )

    if size > 0:

        file = open(output_file, "wb+")

        file.write(evt.result.audio)

        file.close()

translator.synthesizing.connect(
    synthesis_callback
)

print(f"Speak now ({source_language})")

translation_results = translator.recognize_once()

print(translation_results.translations[target_language])

playsound(output_file)
```

---

# Diferencias entre ambos enfoques

| Manual Synthesis | Event-Based Synthesis |
|------------------|----------------------|
| Usa `SpeechSynthesizer`. | No usa `SpeechSynthesizer`. |
| Dos pasos separados. | Todo ocurre automáticamente. |
| Soporta múltiples idiomas. | Solo un idioma destino. |
| Más flexible. | Más simple. |
| Tú controlas la síntesis. | El SDK entrega directamente el audio. |

# Conceptos importantes para el examen

- `SpeechTranslationConfig` configura la traducción de voz.
- `TranslationRecognizer` realiza el reconocimiento y la traducción.
- `AudioConfig` define el origen del audio.
- Se puede traducir a **uno o varios idiomas**.
- `translation_results.text` contiene el texto reconocido.
- `translation_results.translations` contiene las traducciones.
- **Manual Synthesis** combina `TranslationRecognizer` + `SpeechSynthesizer`.
- **Event-Based Synthesis** entrega directamente el audio mediante el evento `Synthesizing`.
- **Event-Based Synthesis solo soporta traducción 1:1**.
- Manual Synthesis permite elegir una voz distinta para cada idioma traducido.
