# Use the Speech to Text API

## ¿Qué es?
API de **Azure Speech** que convierte **audio** en **texto** (*Speech-to-Text*).

## Componentes principales

### SpeechConfig
Configura la conexión con el recurso Foundry.

Contiene:
- Endpoint.
- Subscription Key.

### AudioConfig
Define la fuente del audio.

Puede ser:
- 🎤 Micrófono (por defecto).
- 📁 Archivo de audio.

### SpeechRecognizer
Cliente que consume la **Speech to Text API**.

Responsabilidad:
- Enviar el audio.
- Obtener la transcripción.

## Flujo de funcionamiento

```text
SpeechConfig
      │
      ▼
AudioConfig
      │
      ▼
SpeechRecognizer
      │
      ▼
RecognizeOnceAsync()
      │
      ▼
Speech to Text API
      │
      ▼
SpeechRecognitionResult
```

## Método principal

### `RecognizeOnceAsync()`

- Transcribe **una única intervención** de voz.
- Devuelve un **SpeechRecognitionResult**.

## SpeechRecognitionResult

Propiedades importantes:

- `Text` → texto transcrito.
- `Reason` → resultado de la operación.
- `Duration`
- `OffsetInTicks`
- `Properties`
- `ResultId`

## Valores de `Reason`

| Valor | Significado |
|--------|-------------|
| `RecognizedSpeech` | ✅ Transcripción exitosa. |
| `NoMatch` | ⚠️ Se procesó el audio, pero no se detectó voz. |
| `Canceled` | ❌ Ocurrió un error (consultar `CancellationReason`). |

## Flujo de uso

1. Crear `SpeechConfig`.
2. Crear `AudioConfig`.
3. Crear `SpeechRecognizer`.
4. Ejecutar `RecognizeOnceAsync()`.
5. Revisar `Reason`.
6. Leer el texto desde `Text`.

## Ejemplo

```text
audio.wav
     │
     ▼
AudioConfig
     │
     ▼
SpeechRecognizer
     │
     ▼
RecognizeOnceAsync()
     │
     ▼
SpeechRecognitionResult
     │
     ├── Reason = RecognizedSpeech
     └── Text = "Hello everyone..."
```

## Idea clave
- **SpeechConfig** → conexión al servicio.
- **AudioConfig** → origen del audio.
- **SpeechRecognizer** → realiza la transcripción.
- **RecognizeOnceAsync()** → transcribe una sola intervención.
- El resultado se obtiene desde **SpeechRecognitionResult**, especialmente en `Text` y `Reason`.
