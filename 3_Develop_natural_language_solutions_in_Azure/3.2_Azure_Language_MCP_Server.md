# Azure Language MCP Server

## ¿Qué es?
Servidor **MCP (Model Context Protocol)** que expone las capacidades de **Azure Language** como herramientas que un agente de IA puede descubrir y utilizar dinámicamente.

## ¿Qué es MCP?
Protocolo abierto para conectar agentes de IA con herramientas y servicios externos.

### Arquitectura

- **Host:** ejecuta el agente (Microsoft Foundry, aplicación propia, etc.).
- **Client:** administra la comunicación con servidores MCP.
- **Server:** expone herramientas, recursos y prompts.

```text
Usuario
    │
    ▼
Host (Agente)
    │
    ▼
MCP Client
    │
    ▼
MCP Server
    │
    ▼
Azure Language
```

## Dynamic Tool Discovery
El agente **no necesita conocer las herramientas de antemano**.

Funcionamiento:
1. Consulta el servidor MCP.
2. Obtiene el catálogo de herramientas.
3. Elige automáticamente la más adecuada.

### Ventajas
- 🔄 Agregar o actualizar herramientas sin modificar el agente.
- 📈 Mayor escalabilidad y mantenimiento.
- 🤖 Selección automática de herramientas.

## Herramientas disponibles

| Herramienta | Función |
|-------------|---------|
| **Language Detection** | Detecta el idioma. |
| **Named Entity Recognition (NER)** | Identifica personas, lugares, organizaciones, fechas, etc. |
| **PII Redaction** | Detecta y oculta información personal. |
| **Text Analytics for Health** | Extrae entidades médicas (diagnósticos, medicamentos, síntomas, etc.). |

> Azure Language también incluye otras funciones heredadas como análisis de sentimiento, resumen y extracción de frases clave.

## Selección automática de herramientas

Flujo:

```text
Usuario
    │
    ▼
Agente analiza el prompt
    │
    ▼
Consulta herramientas MCP
    │
    ▼
Selecciona una o varias
    │
    ▼
Azure Language procesa
    │
    ▼
Agente genera la respuesta
```

### Ejemplo

Pregunta:

> "¿En qué idioma está este artículo y qué personas se mencionan?"

El agente puede invocar automáticamente:

- Language Detection
- Named Entity Recognition

## Endpoint del servidor

```text
https://{foundry-resource-name}.cognitiveservices.azure.com/language/mcp?api-version=2025-11-15-preview
```

Reemplazar:

- `{foundry-resource-name}` → nombre del recurso de Azure Foundry o Azure Language.

## Tipos de servidor MCP
- 🌐 **Remoto:** endpoint hospedado en Azure.
- 💻 **Local:** puede ejecutarse en tu propia infraestructura.

## Idea clave
- **MCP** permite que un agente descubra y utilice herramientas dinámicamente.
- El **Azure Language MCP Server** publica las capacidades de Azure Language como herramientas MCP.
- El agente decide automáticamente qué herramienta usar según el prompt, sin necesidad de lógica de enrutamiento manual.
