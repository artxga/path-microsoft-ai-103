# Azure Speech in Microsoft Foundry Tools

## ¿Qué es?
Servicio de **Azure Speech** integrado en **Microsoft Foundry** para agregar capacidades de voz a aplicaciones y agentes.

## Casos de uso
- 🎙️ **Speech-to-Text (STT):** transcribir llamadas, reuniones o audios.
- 🔊 **Text-to-Speech (TTS):** leer mensajes, correos o generar asistentes con voz.

## Requisitos
- Crear un **Microsoft Foundry Resource** en Azure.
- Obtener:
  - **Endpoint**
  - **Key**

## Formas de consumo
- REST (JSON).
- SDKs oficiales:
  - Python
  - C#
  - JavaScript
  - Otros.

## SpeechConfig

Es el objeto que establece la conexión con **Azure Speech**.

Contiene:
- **Subscription Key**
- **Endpoint**

Ejemplo conceptual:

```python
speech_config = SpeechConfig(
    subscription=KEY,
    endpoint=ENDPOINT
)
```

## Flujo de funcionamiento

```text
Aplicación
      │
      ▼
SpeechConfig
      │
      ▼
Azure Speech
      │
      ├── Speech-to-Text
      └── Text-to-Speech
      │
      ▼
Resultado
```

## Idea clave
- **Azure Speech** añade capacidades de voz a aplicaciones y agentes.
- La conexión se configura mediante un **SpeechConfig**.
- **SpeechConfig** necesita únicamente el **Endpoint** y la **Key** del recurso Foundry.
- Permite utilizar tanto **Speech-to-Text** como **Text-to-Speech** desde el mismo recurso.
