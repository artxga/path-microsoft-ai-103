# Create a client application that uses an image generation model

Los modelos de generación de imágenes pueden utilizarse desde una aplicación cliente mediante los SDKs de OpenAI o Azure OpenAI.

El flujo consiste en:

1. Enviar un **prompt** descriptivo.
2. El modelo genera la imagen.
3. La imagen se devuelve codificada en **Base64**.
4. Decodificar el Base64.
5. Guardar la imagen como un archivo (por ejemplo, PNG).

---

# Flujo general

```
Prompt
   │
   ▼
Modelo de generación de imágenes
   │
   ▼
Imagen (Base64)
   │
   ▼
Decodificar Base64
   │
   ▼
Guardar archivo (image.png)
```

---

# Generar una imagen

```python
# Generate an image
img_results = client.images.generate(
    model="FLUX.1-Kontext-pro",
    prompt="A robot eating a cheeseburger.",
    n=1,
    size="1024x1024",
)
```

## Parámetros

| Parámetro | Descripción |
|-----------|-------------|
| `model` | Modelo de generación de imágenes. |
| `prompt` | Descripción de la imagen que se desea generar. |
| `n` | Número de imágenes a generar. |
| `size` | Resolución de la imagen. |

---

# Guardar la imagen

El servicio devuelve la imagen en formato **Base64** (`b64_json`).

```python
image_data = base64.b64decode(img_results.data[0].b64_json)

with open("image.png", "wb") as image_file:
    image_file.write(image_data)
```

---

# Explicación

La respuesta contiene una lista de imágenes:

```python
img_results.data
```

Cada imagen incluye:

```python
img_results.data[0].b64_json
```

Ese contenido es una cadena Base64 que representa la imagen.

Se convierte a bytes con:

```python
base64.b64decode(...)
```

Finalmente se guarda como un archivo:

```python
with open("image.png", "wb") as image_file:
    image_file.write(image_data)
```

---

# Flujo completo

```
Prompt
"A robot eating a cheeseburger."
          │
          ▼
client.images.generate(...)
          │
          ▼
Respuesta
          │
          ▼
img_results.data[0].b64_json
          │
          ▼
base64.b64decode(...)
          │
          ▼
image.png
```

---

# Ejemplo completo

```python
import base64

# Generar la imagen
img_results = client.images.generate(
    model="FLUX.1-Kontext-pro",
    prompt="A robot eating a cheeseburger.",
    n=1,
    size="1024x1024",
)

# Decodificar Base64
image_data = base64.b64decode(img_results.data[0].b64_json)

# Guardar imagen
with open("image.png", "wb") as image_file:
    image_file.write(image_data)

print("Imagen guardada como image.png")
```

---

# Idea clave para el examen

- La generación de imágenes se realiza mediante `client.images.generate()`.
- El **prompt** describe la imagen deseada.
- La respuesta devuelve la imagen codificada en **Base64** (`b64_json`).
- Es necesario decodificar el Base64 y guardar el resultado como un archivo (por ejemplo, `image.png`).
- Se pueden generar una o varias imágenes mediante el parámetro `n`.
- El tamaño de salida se controla con el parámetro `size`.
