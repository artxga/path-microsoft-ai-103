# Azure Language in Microsoft Foundry Tools

## ¿Qué es?
Servicio para **analizar texto** y extraer información mediante IA.

## Funcionalidades principales
- 🌍 **Language Detection:** detecta el idioma del texto.
- 🏷️ **Named Entity Recognition (NER):** identifica personas, lugares, organizaciones, fechas, etc.
- 🔒 **PII Extraction:** detecta y puede ocultar información personal (PII).

> También soporta análisis de sentimiento, resumen y extracción de frases clave (funciones heredadas para compatibilidad).

## Requisitos
- Crear un **Microsoft Foundry Resource** en Azure.
- Obtener:
  - **Endpoint**
  - **Key** (o usar Microsoft Entra ID).

## Formas de consumir la API
- REST (JSON).
- SDKs oficiales:
  - Python
  - C#
  - JavaScript
  - Otros.

## Autenticación

### API Key
Utiliza:
- Endpoint del recurso.
- Key del recurso.

```python
credential = AzureKeyCredential(KEY)

client = TextAnalyticsClient(
    endpoint=ENDPOINT,
    credential=credential
)
```

### Microsoft Entra ID (recomendado)
Más seguro para producción.

```python
credential = DefaultAzureCredential()

client = TextAnalyticsClient(
    endpoint=ENDPOINT,
    credential=credential
)
```

## Cliente principal
**TextAnalyticsClient**

Permite llamar a las APIs de Azure Language para analizar texto.

## Flujo de uso
```text
Texto
   │
   ▼
TextAnalyticsClient
   │
   ▼
Azure Language
   │
   ├── Detect Language
   ├── Named Entity Recognition
   └── PII Detection
   │
   ▼
Resultado
```

## Idea clave
- **Azure Language** analiza texto mediante IA.
- Se utiliza a través de **TextAnalyticsClient**.
- La autenticación puede hacerse con **API Key** o **Microsoft Entra ID** (recomendado).
- Funciones principales: **Language Detection**, **NER** y **PII Extraction**.
