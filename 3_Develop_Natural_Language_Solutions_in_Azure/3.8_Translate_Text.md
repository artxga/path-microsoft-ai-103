# Azure Translator en Microsoft Foundry Tools

## ¿Qué es?

**Azure Translator** permite traducir texto entre **más de 90 idiomas**.

Además de la traducción tradicional, también soporta:

- Traducción de texto (Text Translation).
- Transliteración (cambiar el sistema de escritura sin cambiar el idioma).
- Traducción de documentos (sincrónica y asíncrona).
- Modelos personalizados para vocabulario específico.
- Traducción usando modelos LLM o el modelo clásico de Translator.

> En este módulo el enfoque principal es **Text Translation**.

# Conectarse a Azure Translator

Se utiliza `TextTranslationClient`.

Puede conectarse mediante:

- Endpoint global
- Endpoint regional
- Endpoint del recurso Foundry

## Ejemplo

```python
from azure.core.credentials import AzureKeyCredential
from azure.ai.translation.text import *

key_credential = AzureKeyCredential("FOUNDRY_KEY")

# Conectarse mediante endpoint
client = TextTranslationClient(
    credential=key_credential,
    endpoint="FOUNDRY_ENDPOINT"
)

# O mediante región
client = TextTranslationClient(
    credential=key_credential,
    region="FOUNDRY_REGION"
)
```

# Obtener idiomas soportados

Azure Translator soporta **más de 90 idiomas**.

Puedes obtener la lista completa dinámicamente.

## Ejemplo

```python
languages = client.get_supported_languages(scope="translation")

print("{} languages supported:".format(len(languages.translation)))

for language in languages.translation.keys():
    print(
        languages.translation[language].name +
        " (" + language + ")"
    )
```

Salida:

```text
Afrikaans (af)
Arabic (ar)
English (en)
Spanish (es)
Japanese (ja)
...
```

Esto evita mantener listas estáticas de idiomas.

# Traducir texto

La operación principal es:

```python
translate()
```

Recibe:

- Uno o varios textos (`InputTextItem`)
- Uno o varios idiomas destino
- Opcionalmente el idioma origen

Si no se especifica el idioma origen, Translator lo detecta automáticamente.

---

## Ejemplo

```python
input_text_elements = [
    InputTextItem(text="Hola"),
    InputTextItem(text="こんにちは")
]

translation_results = client.translate(
    body=input_text_elements,
    to_language=["fr", "en"]
)

idx = 0

for translation in translation_results:

    input_text = input_text_elements[idx].text
    idx += 1

    sourceLanguage = translation.detected_language

    for translated_text in translation.translations:

        print(
            f"'{input_text}' was translated "
            f"from {sourceLanguage.language} "
            f"to {translated_text.to} "
            f"as '{translated_text.text}'."
        )
```

Resultado

```text
'Hola' was translated from es to fr as 'Bonjour'.

'Hola' was translated from es to en as 'Hello'.

'こんにちは' was translated from ja to fr as 'Bonjour'.

'こんにちは' was translated from ja to en as 'Hello'.
```

# Detección automática del idioma

No es necesario indicar:

```python
from_language="es"
```

Azure Translator detecta automáticamente el idioma.

En el resultado se obtiene:

```python
translation.detected_language.language
```

Por ejemplo:

```text
es
ja
en
fr
```

# Traducir a varios idiomas

Se puede traducir un mismo texto a múltiples idiomas en una sola llamada.

Ejemplo:

```python
client.translate(
    body=input_text_elements,
    to_language=[
        "es",
        "en",
        "fr",
        "de",
        "it"
    ]
)
```

Esto evita múltiples llamadas a la API.

# Transliteración

La transliteración **NO traduce**.

Solo cambia el sistema de escritura.

Ejemplo:

```text
こんにちは
```

↓

```text
Kon'nichiwa
```

Sigue siendo japonés.

Solo cambió el alfabeto.

---

# Usar `transliterate()`

Debe indicarse:

- idioma
- escritura origen
- escritura destino

## Ejemplo

```python
source_text = "こんにちは"

input_text_elements = [
    InputTextItem(text=source_text)
]

transliteration_results = client.transliterate(
    body=input_text_elements,
    language="ja",
    from_script="Jpan",
    to_script="Latn"
)

for transliteration in transliteration_results:

    sourceScript = transliteration.script
    targetScript = transliteration.text

    print(
        f"'{source_text}' "
        f"was transliterated "
        f"into {sourceScript} "
        f"as {targetScript}."
    )
```

Resultado

```text
'こんにちは' was transliterated into Latn as Kon'nichiwa.
```

---

# Diferencia entre traducción y transliteración

## Traducción

```text
Hola
↓

Hello
```

Cambia el idioma.

---

## Transliteración

```text
こんにちは
↓

Kon'nichiwa
```

No cambia el idioma.

Solo cambia el sistema de escritura.


# Conceptos importantes para el examen

- Azure Translator soporta **más de 90 idiomas**.
- Puede usarse desde **REST API** o **SDK**.
- Se conecta mediante **endpoint** o **región**.
- `TextTranslationClient` es el cliente principal.
- `translate()` traduce uno o varios textos a uno o varios idiomas.
- El idioma origen puede detectarse automáticamente.
- `get_supported_languages()` devuelve los idiomas disponibles.
- `transliterate()` cambia el sistema de escritura, **no el idioma**.
- Azure Translator puede utilizar el modelo clásico o modelos **LLM** para la traducción.
