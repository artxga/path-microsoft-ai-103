# Transcribe Speech (Speech-to-Text)

## ¿Qué es?
Convierte **audio** en **texto** utilizando un modelo de transcripción.

## Modelos compatibles
- `gpt-4o-transcribe`
- `gpt-4o-mini-transcribe`
- `gpt-4o-transcribe-diarize` *(identifica diferentes hablantes).*

> La disponibilidad depende de la región de Azure.

## Flujo de funcionamiento

```text
Archivo de audio
       │
       ▼
AzureOpenAI Client
       │
       ▼
Modelo de transcripción
       │
       ▼
Texto transcrito
```

## Pasos de uso

1. Crear un **AzureOpenAI Client**.
2. Conectarse al **endpoint** del recurso Foundry.
3. Abrir el archivo de audio (`.mp3`, etc.).
4. Llamar a `audio.transcriptions.create()`.
5. Obtener la transcripción en texto.

## Componentes principales

### AzureOpenAI
Cliente para interactuar con modelos de Azure OpenAI.

### audio.transcriptions.create()
Método que envía el audio al modelo para transcribirlo.

Parámetros principales:
- `model` → despliegue del modelo.
- `file` → archivo de audio.
- `response_format` → formato de salida (ej. `"text"`).

## Ejemplo de flujo

```text
speech.mp3
      │
      ▼
AzureOpenAI Client
      │
      ▼
gpt-4o-transcribe
      │
      ▼
"Hello everyone, welcome..."
```

## Idea clave
- **Speech-to-Text** transforma audio en texto.
- Se utiliza mediante **AzureOpenAI**.
- La transcripción se realiza con `audio.transcriptions.create()`.
- Modelos disponibles: **gpt-4o-transcribe**, **gpt-4o-mini-transcribe** y **gpt-4o-transcribe-diarize**.
