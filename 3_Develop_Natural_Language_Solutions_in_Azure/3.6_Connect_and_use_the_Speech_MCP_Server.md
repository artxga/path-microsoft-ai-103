# Connect and Use the Azure Speech MCP Server

## Objetivo
Conectar el **Azure Speech MCP Server** a un agente de **Microsoft Foundry** para que pueda transcribir audio (**Speech-to-Text**) y generar voz (**Text-to-Speech**).

## 1. Configurar Azure Blob Storage

El Speech MCP Server necesita un contenedor para almacenar archivos de audio.

Pasos:
1. Crear un **Azure Storage Account**.
2. Crear un **Blob Container** (ej. `files`).
3. Generar una **SAS URL** con permisos:
   - Read
   - Add
   - Create
   - Write
   - List
4. Guardar la SAS URL de forma segura.

> ⚠️ La **SAS URL** es un secreto y debe tener el menor tiempo de expiración posible.

## 2. Crear el agente

Requisitos:
- Proyecto en **Microsoft Foundry**.
- Modelo desplegado (ej. **GPT-4.1**).

Ejemplo de instrucciones:

```text
You are an AI agent that uses the Azure AI Speech tool
to transcribe and generate speech.
```

## 3. Conectar el Azure Speech MCP Server

Desde **Foundry Portal → Tools**:

1. Connect a tool.
2. Seleccionar **Azure Speech in Foundry Tools**.
3. Configurar:

- **Foundry Resource Name**
- **Ocp-Apim-Subscription-Key**
- **X-Blob-Container-Url** (SAS URL)

4. Asociar la herramienta al agente.

Resultado:
- El agente obtiene acceso a **Speech-to-Text** y **Text-to-Speech**.

## 4. Probar en Agent Playground

### Text-to-Speech

Ejemplo:

```text
Generate "To be or not to be..."
as speech.
```

Resultado:
- El agente genera el audio.
- Lo guarda en Blob Storage.
- Devuelve un enlace al archivo.

### Speech-to-Text

Ejemplo:

```text
Transcribe the file at
https://.../meeting.wav
```

Resultado:
- El agente transcribe el audio.
- Devuelve el texto.

## 5. Personalizar la voz

Se puede indicar en el prompt:

- 🗣️ Voz (`en-US-JennyNeural`)
- 🌍 Idioma (`es-ES`)
- 💡 Phrase Hints
- 🤬 Filtro de lenguaje ofensivo

Ejemplo:

```text
Synthesize:

"Better a witty fool..."

using voice

"en-GB-SoniaNeural"
```

## 6. Crear una aplicación cliente

SDKs:

- `azure-ai-projects`
- `azure-identity`

Flujo:

```text
AIProjectClient
       │
       ▼
OpenAI Client
       │
       ▼
responses.create()
       │
       ▼
Speech Agent
       │
       ▼
Speech MCP Server
```

El agente se referencia mediante `agent_reference`.

## 7. Conectar el MCP Server desde código

Puede configurarse mediante **MCPTool**.

Se define:

- `server_label`
- `server_url`
- `require_approval`

Luego se pasa al crear el agente desde el SDK.

## Flujo completo

```text
Usuario
    │
    ▼
Speech Agent
    │
    ▼
Speech MCP Server
    │
    ├── Speech-to-Text
    └── Text-to-Speech
    │
    ▼
Azure Speech
    │
    ▼
Azure Blob Storage
    │
    ▼
Texto o enlace al audio
```

## Idea clave
- El **Azure Speech MCP Server** añade capacidades de **Speech-to-Text** y **Text-to-Speech** a un agente.
- Requiere **Azure Blob Storage** y una **SAS URL** para almacenar o leer archivos de audio.
- Puede configurarse desde el **Foundry Portal** o mediante **`MCPTool`** en código.
- Los prompts pueden personalizar la voz, idioma, pistas de reconocimiento (*phrase hints*) y el filtrado de lenguaje ofensivo.
