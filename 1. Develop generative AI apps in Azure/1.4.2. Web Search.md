# Web Search Tool

## ¿Qué es?

**Web Search** es una herramienta que permite a un modelo consultar **información actualizada de Internet** antes de generar una respuesta.

En lugar de depender únicamente de los datos con los que fue entrenado, el modelo puede buscar contenido reciente en la web y utilizarlo para responder con mayor precisión. :contentReference[oaicite:0]{index=0}

---

# ¿Por qué usar Web Search?

Los modelos tienen un conocimiento limitado a su fecha de entrenamiento.

Con **Web Search** pueden acceder a:

- Noticias recientes.
- Eventos actuales.
- Datos en tiempo real.
- Información que cambia con frecuencia.

Esto reduce el riesgo de responder con información desactualizada. :contentReference[oaicite:1]{index=1}

---

# ¿Cómo funciona?

Cuando un usuario realiza una consulta:

1. El usuario envía un **prompt**.
2. El modelo determina si necesita información actualizada.
3. Si es necesario, utiliza la herramienta **Web Search**.
4. Se realiza una búsqueda en Internet.
5. Los resultados relevantes se envían al modelo.
6. El modelo genera una respuesta basada en esa información e incluye referencias a las fuentes cuando corresponde. :contentReference[oaicite:2]{index=2}

# Capacidades

## Obtener información actualizada

Permite responder preguntas sobre:

- Noticias.
- Clima.
- Eventos.
- Deportes.
- Economía.
- Tecnología.
- Información pública reciente.

---

## Fundamentar respuestas (Grounding)

El modelo utiliza los resultados encontrados para **respaldar sus respuestas con información de la web**, en lugar de responder únicamente con conocimiento interno. :contentReference[oaicite:3]{index=3}

---

## Citar fuentes

Las respuestas pueden incluir referencias a las páginas utilizadas, lo que facilita verificar la información. :contentReference[oaicite:4]{index=4}

---

# Casos de uso

- Consultar el clima actual.
- Buscar noticias recientes.
- Verificar información de empresas o productos.
- Obtener resultados deportivos.
- Consultar documentación o contenido publicado recientemente.

---

# Ejemplo

**Prompt**

> ¿Cuáles fueron las principales noticias sobre inteligencia artificial esta semana?

**Proceso**

1. El modelo detecta que la información debe ser reciente.
2. Ejecuta una búsqueda en la web.
3. Analiza los resultados más relevantes.
4. Genera un resumen con referencias a las fuentes consultadas.

---

# Beneficios

- Información actualizada.
- Respuestas más precisas.
- Reduce respuestas desactualizadas.
- Permite citar fuentes de información.
- Amplía el conocimiento del modelo más allá de su entrenamiento.

---

# Consideraciones

- Depende de la disponibilidad de información pública en Internet.
- El uso de Web Search puede generar costos adicionales según el servicio utilizado.
- La información enviada a los servicios de búsqueda puede salir de la región de cumplimiento de Azure, por lo que debe considerarse en escenarios con requisitos de residencia de datos. :contentReference[oaicite:5]{index=5}

---

# Resumen

- **Web Search** permite que un modelo consulte información actualizada en Internet antes de responder.
- El modelo decide automáticamente cuándo utilizar la herramienta.
- Los resultados obtenidos sirven para fundamentar las respuestas y pueden incluir citas a las fuentes.
- Es ideal para consultas que requieren información en tiempo real o que cambia con frecuencia, como noticias, clima y eventos.
